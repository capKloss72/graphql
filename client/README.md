# GraphQL Client

A client application for consuming GraphQL APIs.

## Setup

```bash
cd client
npm install
```

## Running

```bash
npm start
```

## Development

```bash
npm run dev
```

## Features

- Basic GraphQL client setup
- Example query functionality
- Ready to integrate with popular GraphQL clients like Apollo Client or URQL

## Configuration

The client is configured to connect to the GraphQL server at `http://localhost:9000/graphql` by default.
